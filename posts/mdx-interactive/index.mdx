# MDX 交互式组件示例

MDX 让我们能够在 Markdown 中无缝使用 React 组件，创造出丰富的交互式内容体验。

## 1. 基础交互组件

### 计数器
这是一个简单的计数器组件，展示了状态管理：

<Counter />

### 切换开关
一个带有动画效果的切换开关：

<Toggle />

## 2. 数据可视化

### 实时图表
这个组件展示了一个实时更新的简单图表：

<LiveChart />

### 进度条动画
一个可以手动控制的进度条：

<ProgressBar />

## 3. 用户输入组件

### 实时搜索
输入内容并查看实时搜索结果：

<LiveSearch />

### 评分组件
一个星级评分组件：

<StarRating />

## 4. 动画和视觉效果

### 粒子动画
一个简单的粒子动画效果：

<ParticleAnimation />

### 颜色选择器
一个交互式的颜色选择工具：

<ColorPicker />

## 5. 游戏化组件

### 猜数字游戏
一个简单的猜数字小游戏：

<GuessNumber />

### 记忆卡片游戏
测试你的记忆力：

<MemoryGame />

## 总结

MDX 的强大之处在于能够将静态内容与动态交互无缝结合。这些组件展示了：

- **状态管理**：使用 React hooks 管理组件状态
- **事件处理**：响应用户交互
- **动画效果**：使用 CSS 和 JavaScript 创建流畅动画
- **数据可视化**：实时展示数据变化
- **游戏化元素**：增加内容的趣味性

通过 MDX，我们可以创建真正引人入胜的技术文档和博客文章。
