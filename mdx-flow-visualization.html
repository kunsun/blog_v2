你的配置解析

remarkPlugins 配置：

remarkPlugins: [ remarkSmartpants, // 智能标点：将 -- 转为 —，'' 转为 "" remarkGfm, // GitHub Flavored Markdown（表格、删除线等） [remarkMdxEvalCodeBlock, filename] // 自定义代码块处理（传入文件名参数） ]

rehypePlugins 配置：

rehypePlugins: [ [rehypePrettyCode, { theme, // 代码高亮主题（dark-plus + github-light） keepBackground: false // 不保留主题背景色，使用自定义样式 }] ]

处理示例

输入：

## 标题 这是 "智能引号" 和 -- 破折号的例子。 | 表格 | 示例 | |------|------| | 行1 | 数据 | ```javascript function hello() { console.log("Hello World!"); } ```

经过 remarkPlugins 处理后：

- remarkSmartpants: "智能引号" → "智能引号", -- → — - remarkGfm: 表格语法被正确解析 - remarkMdxEvalCodeBlock: 代码块被标记和处理

经过 rehypePlugins 处理后：

- rehypePrettyCode: JavaScript 代码被添加语法高亮 - 生成带有 CSS 类名的 HTML 结构 - 应用主题颜色（支持深色/浅色模式）
关键点：插件按数组顺序执行，每个插件处理前一个插件的输出结果。remark 阶段处理内容结构，rehype 阶段处理视觉呈现。
